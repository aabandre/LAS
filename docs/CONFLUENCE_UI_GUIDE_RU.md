h1. Local Admin Scanner — инструкция для Confluence (RU)

h2. Назначение
Инструмент для поиска и анализа локальных привилегированных учётных записей на хостах домена, с возможностью массовой ремедиации.

h2. 1. Главная страница (/)

h3. 1.1 Кнопки
|| Кнопка || Назначение ||
| Запустить | Старт сканирования |
| Остановить | Остановка текущего запуска |
| Сохранить | Сохранение настроек в браузере |
| Загрузить | Загрузка сохранённых настроек |
| Stable Fast / Надёжный быстрый | Применение пресета |
| Сводка → | Переход к аналитике |

h3. 1.2 Параметры (кратко)
|| Параметр || Описание ||
| Контроллер домена / GC/DC | Источники AD |
| Domain / NetBIOS | Контекст домена |
| Пользователь / Пароль | Основная УЗ сканирования |
| Логин/пароль для серверов | Отдельная УЗ для server-target |
| Таймауты / лимиты | Баланс скорости и стабильности |
| Разрешённые админы | Исключения из риск-детекта |
| Доп. алиасы доменов | Учитываются при risk-мэтчинге |
| Include/Exclude | Маски охвата сканирования |
| Фильтр по ОС | Отбор по ОС |

h3. 1.3 Runtime-опции
* *Разворачивать доменные группы через LDAP* — по умолчанию выключено (замедляет скан).
* RPC/WMI fallback — включать по необходимости.
* debug-log — для диагностики.

h2. 2. Прогресс
Показывает KPI скана: просканировано, всего, успех, ошибки, участники, скорость, ETA.

h2. 3. Сводка (/summary)

h3. 3.1 Верхняя панель
|| Элемент || Назначение ||
| Печать | Экспорт в печатный вид |
| Экспорт отфильтр. админов (CSV) | CSV по текущему фильтру |
| Загрузить | Открытие прошлой сводки |
| Пересчитать | Перестроение агрегатов |
| Скан 1 / Скан 2 / Сравнить | Сравнение двух запусков |

h3. 3.2 Вкладка «Админы»
* Поиск, фильтры, сортировка, плотность отображения, пагинация.
* Выделение строк чекбоксами.
* *В очередь выбранное* — массовое добавление в remediation queue.
* *В очередь* в строке — точечное добавление.

h3. 3.3 Очередь исправления
|| Поле/кнопка || Назначение ||
| DOMAIN / User / Password | Опциональная авторизация для Run now |
| Выполнить | Выполнить удаление через API |
| Экспорт .ps1 | Сгенерировать скрипт исправления |
| Очистить | Очистить очередь |

Примечание: если поля авторизации пусты — используется контекст по умолчанию.

h3. 3.4 Остальные вкладки
* *Риск* — быстрые фильтры по severity.
* *Чистые* — список безрисковых хостов.
* *Карта* — тепловая карта учёток по хостам.
* *Сравнение* — дельта двух запусков.
* *История* — список файлов результатов.

h2. 4. Рекомендации
* Использовать выделенную сервисную УЗ с минимальными правами.
* Для remediaiton (Run now) использовать повышенную УЗ только при необходимости.
* На больших доменах держать LDAP-экспанд выключенным по умолчанию.
* Перед запуском ремедиации проверять очередь вручную.
